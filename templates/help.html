How to Use the Finance App

💼 Accounts

  • Each user has a default account that cannot be deleted.
  • You can create as many accounts as you want (e.g., Bank, Wallet, Cash).
  • If you delete an account that has transactions, you'll be asked whether to:
    • ❌ Delete the transactions
    • 🔁 Move them to another account of your choice

📂 Categories

  • You start with 5 default categories (Food, Rent, etc.).
  • You can add, edit, and delete your own categories.
  • If no category is selected for a transaction, it's assigned to "Uncategorized".

💸 Transactions

  • Use the Add Transaction button to record income or expenses.
  • Each transaction belongs to an account and optionally a category.
  • You can edit or delete transactions at any time.

📊 Dashboard

  • Shows your total income, expenses, and net for the selected month.
  • Use the month picker to view past activity.
  • A "Back to Current Month" button will appear when you're viewing older months.

👤 Account & Login

  • Only authenticated users can access financial data.
  • You can log in and out from the navigation bar.
