How to Use the Personal Finance Tracker

Managing Accounts

  • Each user has a default account which cannot be deleted.
  • You can create additional accounts (e.g., Bank, Wallet, Cash) as needed.
  • If you delete an account that has transactions, you'll be prompted to:

Organizing Categories

  • Start with a few default categories like Food or Rent.
  • You can add, edit, or remove categories at any time.
  • If no category is selected for a transaction, it will be labeled as "Uncategorized".

Recording Transactions

  • Use the Add Transaction button to log income or expenses.
  • Each transaction must be linked to an account, and optionally to a category.
  • You can edit or delete any transaction from the list.

Analyzing with the Dashboard

  • The dashboard shows a summary of your income, expenses, and net balance by month.
  • Use filters to view specific months or accounts.
  • Visual charts help you understand your financial trends.
  • Click Download PDF to export a summary report.

Account & Access

  • You must be logged in to access your dashboard and data.
  • Login and logout options are available in the navigation bar.
  • Your data is stored securely and is only visible to you.
