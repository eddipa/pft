How to Use the Finance App

Accounts

  • Each user has a default account that cannot be deleted.
  • Create as many accounts as you like (e.g., Bank, Wallet, Cash).
  • If you delete an account that has transactions, you'll be asked to:

    • üîÅ Move them to another account

Categories

  • Start with 5 default categories (e.g., Food, Rent).
  • Create, edit, or delete your own categories anytime.
  • Uncategorized transactions will default to "Uncategorized".

Transactions

  • Click Add Transaction to record income or expenses.
  • Each transaction must belong to an account and optionally a category.
  • You can edit or delete any transaction.

Dashboard

  • View income, expenses, and net balance for each month.
  • Use the month picker to browse financial history.
  • A "Back to Current Month" button helps you return quickly.

Account & Login

  • Only logged-in users can access financial data.
  • Login/logout available in the navigation bar.
